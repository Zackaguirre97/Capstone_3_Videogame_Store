# Video Game Store

This project is a full stack video game store web application built as part of the Year Up United / Pluralsight program. The application demonstrates core backend and frontend concepts including authentication, database interaction, and dynamic UI updates.

## Features

* User authentication and authorization with JWT
* Product browsing with category and price filtering
* User profile management with saved shipping information
* Shopping cart with add, remove, and clear functionality
* Order checkout with order and line item persistence
* Persistent dark mode using local storage
* Responsive UI built with JavaScript, HTML, and CSS

## Tech Stack

* Backend: Java, Spring Boot, JDBC, MySQL
* Frontend: JavaScript, HTML, CSS, Axios
* Authentication: JWT
* Database: MySQL

## Project Structure

* Backend handles users, profiles, products, carts, and orders
* Frontend communicates with the API using Axios
* Data is persisted using DAO patterns
* UI state (cart, dark mode) is managed client side

## Status

Core functionality is complete and meets all required project phases. Additional enhancements and UI refinements may be added in the future.
